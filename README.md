# Bruxas Burger
 
 Remodelação do site da hamburgueria Bruxas Burger (http://bruxasburger.com.br/).
 
## Tecnologias usadas:
 - PHP
 - AJAX
 - JQuery
 - Javascript
 - HTML + CSS
 - Bootstrap
 - SQL (banco de dados MySQL, gerenciado pelo PHPMyAdmin)
 - Composer
 - PSRs

A aplicação é feita usando o padrão MVC, pois conta com um gerenciador de cardápio e filiais da empresa.
O frontend foi feito com Bootstrap, servindo muito bem para que o site ficasse responsivo.

O site conta com pesquisas dessas mesmas filiais e itens do cardápio sem atualizar a página, com consulta feita ao PHP usando AJAX.
